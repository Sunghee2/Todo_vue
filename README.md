```

```
> 빌드를 하게 되면 `dist`폴더가 생기게 되는데 이 폴더를 나중에 배포하게 됨

`asset/` : css, 이미지들
`components/` : 모든 페이지에서 사용하는 공통적인 컴포넌트
`views` : 각 화면의 컴포넌트
